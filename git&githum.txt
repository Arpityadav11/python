ðŸ”¹ Git
> Git is a **distributed version control system (VCS)** used to track changes in source code during software development.  
It allows multiple developers to collaborate, manage versions, and maintain history of code changes efficiently.

**Key Features:**
- Local and remote repositories
- Branching and merging
- Commit history and rollback
- Conflict resolution

ðŸ”¹ GitHub
> GitHub is a **cloud-based platform** that hosts Git repositories and provides collaboration tools for developers.  
It enables code sharing, version control, and team workflows through features like pull requests, issues, and GitHub Actions.

**Key Services:**
- Remote repository hosting
- Pull requests and code reviews
- Issue tracking and project boards
- CI/CD with GitHub Actions
